rigz

Drop-in gzip replacement. Parallel compression in Rust.

2.6× faster than gzip

Install

# Build from source $ cargo install --path . # Or download a release $ curl -L https://github.com/jackdanger/rigz/releases/latest/download/rigz-$(uname -m) -o rigz

Usage

Works exactly like gzip. Same flags, same behavior.

rigz file.txt → file.txt.gz
rigz -d file.txt.gz decompress
tar cf - dir/ | rigz > archive.tar.gz pipe
rigz -1 / -9 fast / best

Performance

Tested on 622.2MB. Right is faster. Hover for times.
